Beneficios

Con la herramienta Quicktype como un conversor de datos podemos transformar un JSON que es tomado como input para producir un modelo de los objetos que necesitemos que esten construidos en nuestro codigo.



  • Nos puede ayudar a crear objetos que nesesitemos que esten serializados o deserializados.

  • Poder agilizar el desarrollo de APIs.

  • Quicktype trabaja con Postman.

  • En caso de que se tenga una coleccion grande en Postman estas pueden ser exportadas en Quicktype.

  • El proceso es agil. Una vez elegimos el lenguaje de programacion que queramos, todo el codigo generado es mostrado inmediatamente en la pagina de Quicktype.

  • Tambien podemos pasar todos los datos generados en la página web e importarlo a VSCode.

    • Comentario del grupo

    Hay casos donde necesitas el codigo de alguna api de la que estas extrayendo informacion, o tambien de algun objeto que necesites disponibilizar para un sistema externo.

    Cuando necesitamos instanciar un objeto con los nombres y atributos exactos de nuestro envio o recibo de datos es donde entra el trabajo de esta herramienta.

    Ya que podemos usar Quicktype con VSCode tambien podemos usarlo como librerias y modificar la logica del proceso de conversion para producir resultados mas esperados.

    Quicktype no solo acepta JSON como unica entrada de datos, para datos mas rigurosos podemos usar JSON Schema o GraphQL. Aunque actualmente estan trabajando para aceptar mas tipos de datos como CSV, XML, HTML, etc.
