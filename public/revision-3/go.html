¿Qué es Go?

El lenguaje de programación Go es un proyecto de código abierto para hacer que los programadores sean más productivos. Go es expresivo, conciso, limpio y eficiente. Sus mecanismos de concurrencia facilitan la escritura de programas que aprovechan al máximo las máquinas multinúcleo y en red, mientras que su novedoso sistema de tipos permite la construcción de programas modulares y flexibles. Go compila rápidamente en código de máquina pero tiene la comodidad de la recolección de elementos no utilizados y el poder de la reflexión en tiempo de ejecución. Es un lenguaje compilado rápido, tipificado estáticamente que se siente como un lenguaje interpretado y tipificado dinámicamente.

$ rm -rf /usr/local/go && tar -C /usr/local -xzf go1.18.2.linux-amd64.tar.gz

2. Agregue /usr/local/go/bin a la variable de entorno PATH.

Puede hacerlo agregando la siguiente línea a su $HOME/.profile o /etc/profile (para una instalación en todo el sistema):

export PATH=$PATH:/usr/local/go/bin

$ go version

Primeros pasos

Como primer vistazo a cualquier lenguaje de programación haremos el clásico ejemplo de un programa que imprime en pantalla el famoso "Hola mundo!"

Para esto es necesario pegar el siguiente código en nuestro editor de texto de nuestra preferencia:

                    
                      package main
                      import "fmt"
                      // this is a comment
                      func main() {
                          fmt.Println("Hola mundo!")
                      }

Abre una nueva terminal y escribe lo siguiente 

                  
                    go run {nombre de tu archivo}.go

Tipos de datos y variables

En el ejemplo anterior usamos un dato de tipo String para almacenar "Hola mundo!". Los tipos de datos categorizan un grupo de datos relacionados, describen la operacion que pueden hacerse en ellos y definen la forma en la que se almacenan.

Numéricos

Go tiene diferentes tipos de datos para representar números. Generalmente dividimos los numeros en dos tipos: Integers y floating-point numbers

Integers (Enteros)

Para hacerlo facil los enteros (integers) son numeros que no tienen decimal, estos pueden ser o negativos, cero o positivos

Floating Point Numbers (Numeros con decimal)

Asi como los enteros y como su nombre lo dice, este tipo de dato numérico son como los enteros pero con decimales.

Ejemplo de ambos: 

                      
                        package main

                        import "fmt"

                        func main() {
                          fmt.Println("Suma de numeros enteros: 7 + 5 = " , 7+5)
                          fmt.Println("Suma de numeros decimales: 7,7 + 5,3 = " , 7.7 + 5.3)
                        }

Si corrieras este programa el resultado seria el siguiente: 

                    
                      $ go run main.go
                        Suma de numeros enteros: 7 + 5 = 12
                        Suma de numeros decimales: 7,7 + 5,3 = 13

En adicion, Go tiene diferentes operadores:

- Substraccion
+ Adicion
* Multiplicacion
/ Division
% Resto
Strings (Cadenas)

Asi como vimos en el primer ejemplo de "Hola mundo!" un string es una secuencia de caracteres con una longitud definida usada para representar texto: 

                      
                        package main
                        import "fmt"
                        // this is a comment
                        func main() {
                            fmt.Println("Esto es un string!!!")
                        }

Booleans

Un valor boolean es un tipo de dato especial que representa verdadero o falso:

&& and
|| Or
! not 
                      
                        package main
                        import "fmt"
                        // this is a comment
                        func main() {
                          fmt.Println(true && true)
                          fmt.Println(true && false)
                          fmt.Println(true || true)
                          fmt.Println(true || false)
                          fmt.Println(!true)
                        }

Correr este programa imprimiria los siguientes resultados: 

                    
                      $ go run main.go
                        true
                        false
                        true
                        true
                        false
                      }
Variables

Una variable es la locacion donde se almacena un valor, con un tipo y un nombre asignado. Vamos a usar el ejemplo de "Hola mundo" pero aplicando esto: 

                    
                      package main

                      import "fmt"

                      func main() {
                        var x string = "Hello World"
                        fmt.Println(x)
                      }

En este caso nombramos la variable como "x". En cuanto a la reglas de como se debe nombrar una variable contamos con que pueden tener letras, numeros, y _ (barra baja)

Constantes

Go tambien soporta variables de tipo constantes, basicamente son valores los cuales no pueden cambiar su valor luego, para este ejemplo usaremos la palabra "const" en vez de la palabra "var" para declarar una variable de tipo numérico: 

                    
                      package main

                      import "fmt"

                      func main() {
                        const x saldoDisponible = 25000
                        fmt.Println(x)
                      }

Estructuras de control

For

Haciendo uso de lo aprendido con variables, suponiendo que queremos hacer un programar que cuente del 1 al 10 podemos hacerlo de la siguiente manera: 

                  
                    package main

                    import "fmt"

                    func main() {
                      fmt.Println(1)
                      fmt.Println(2)
                      fmt.Println(3)
                      fmt.Println(4)
                      fmt.Println(5)
                      fmt.Println(6)
                      fmt.Println(7)
                      fmt.Println(8)
                      fmt.Println(9)
                      fmt.Println(10)
                    }

Pero si tenemos que imprimir una mayor cantidad de numeros podemos facilitar esta tarea gracias a los For loops

Los for loops o bucles for nos permiten repetir declaraciones muchas veces asi:

                  
                    package main

                    import "fmt"

                    func main() {
                      i := 1
                      for i <= 10 {
                        fmt.Println(i)
                        i = i + 1
                      }
                    }

De esta manera mientras i sea menor o igual a 10 nuestro programa entrará nuevamente en el for loop e imprimira el valor de i para luego sumarle 1

If

Similar a las sentencias for los if tienen un bloque de codigo por debajo, la diferencia esta en que la sentencia if decide que que bloque de codigo ejecutar en base a una condicion. Se ve mejor en este ejemplo: 

                  
                    if i % 2 == 0 {
                      fmt.Println("Par")
                    } else {
                      fmt.Println("Impar")
                    }

Notar la palabra else que permite la ejecucion de imprimir en pantalla "impar" en caso de que la variable i sea impar

Arreglos

Los arreglos (o arrays) son elementos enumerados de un mismo tipo con una longitud definida.

                  
                    var x [5]int

x es un ejemplo de un array de tipo entero que esta compuesta de 5 elementos.

                  
                    package main

                    import "fmt"

                    func main() {
                      var x [5]int
                      x[4] = 100
                      fmt.Println(x)
                    }

Deberias ver esto: 

                  
                    [0 0 0 0 100]

Arrays son indexados, el numero de posicionamiento de sus elementos empieza en 0, entonces en el ejemplo accedimos a la posicion 5 y le dimos el valor de 100.

Funciones

Una funcion es una porcion independiente de codigo que ejecuta esa porcion cuando sea llamado, para esto la funcion tiene un nombre seguido de unos parentesis, asi:

                  
                    func main() {
                      xs := []float64{98,93,77,82,83}
                    
                      total := 0.0
                      for _, v := range xs {
                        total += v
                      }
                      fmt.Println(total / float64(len(xs)))
                    }

La nomenclatura de las funciones empieza por la palabra func seguido del nombre de la funcion. Tambien, los parametro o inputs de la funcion se deben agregar dentro de los parentesis seguido del nombre de la funcion..

                  
                    func f(x int) {
                      fmt.Println(x)
                    }
                    func main() {
                      x := 5
                      f(x)
                    }

Tambien hay una palabra reservada para retornar un valor luego de que una funcion se ejecute, para esto usamos la palabra return

                  
                    func f1() int {
                      return 432
                    }

Debemos indicar que tipo de dato se retornara luego de los parentesis

Conclusion

El lenguaje de programación Go nace en el año 2007 y fue desarrollado por los ingenieros Rob Pike, Ken Thompson y Robert Griesemer, es un lenguaje compilado y concurrente al igual que C desarrollado en google, con una sintaxis clara limpia y organizada. Los creadores de Go se inspiraron en la versatilidad y cualidades de otros lenguajes como Python, C++, Java y entre otros.

Go es eficiente, escalable y productivo, intenta combinar la facilidad de programación de un lenguaje interpretado y tipificado dinámicamente con la eficiencia y seguridad de un lenguaje compilado y tipificado estáticamente.

Varios grandes servicios orientados al usuario lo usan, también muchas otras empresas utilizan Go, la lista es muy larga pero algunos de los más conocidos son: BBC, Docker. DropBox, Heroku, Nokia, Twitter, Paypal, Netflix, Uber, Pinterest y obviamente Google.

Es concurrente, hace que se puedan crear threads (hilos) en un mismo procesador para ejecutar procesos paralelamente.

Posee una sintaxis bastante sencilla, cuando declaras una variable la puedes declarar si es privada o pública tan solo con escribir la primera letra en mayúscula o minúscula. También puedes declarar variables sin indicar el tipo de dato y el compilador va a interpretar que tipo de dato es. No se necesita usar la palabra “var” para declarar, ni “String”, simplemente “:=” para indicar que es una variable y que la estás asignando directamente.

Es capaz de compilar en cualquier sistema operativo, pudiendo crear un ejecutable en código máquina para cualquier sistema.

Go es multiparadigma, es decir, puedes realizar programación estructurada, programación orientada a objetos, programación funcional, entre otras. Posee una guía de estilos bastante estructurada, ya que al estan bien definida, solo se debe respetar el estilo sin necesidad de entrar en discusiones de como escribir el código. Go también cuenta con un garbage recollector (recolector de basura).

Con Go se puede trabajar en áreas como Cloud y Networking, creando servicios y microservicios; también creando Scripts para el sistema (programas que corren en línea de comando); automatizaciones para Devops; para realizar backend, es decir, crear APIS REST ya que go incluye su propio servidor HTTP.

Gracias a su eficacia y a su extraordinario rendimiento, aspectos que se reflejan también en las aplicaciones que se programan con él, Go es una interesante opción especialmente en los ámbitos de los negocios y los servidores.

Es un lenguaje que está creciendo rápidamente, en la actualidad es uno de los más demandados por la industria. Según la encuesta de Hired Go es el lenguaje de programación con mayor demanda laboral para el 2020.

Sin embargo, podemos mencionar que este lenguaje de programación no cuenta con una documentación extensa en cuanto a manuales, procedimientos, libros e informes. Por otra parte, no es posible implementar herencias si no instanciamos de un type.

Por ser un lenguaje de programación que no cuenta con tanta antigüedad como Java, PHP, C#, entre otros, su comunidad aún sigue siendo pequeña y se encuentra en crecimiento.

A pesar de sus desventajas, Go sigue siendo un lenguaje funcional para los casos ya mencionados y al ser un lenguaje en crecimiento podemos esperar una buena evolución para el futuro, por lo que es una excelente opción para aprenderlo hoy en día.
